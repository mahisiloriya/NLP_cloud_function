# NLP_cloud_function
# 📝 User Authentication & NLP Project 🚀  

A simple Python project that allows user registration and login, followed by three NLP-powered functionalities:  
1️⃣ **Sentiment Analysis** – Classifies text as **positive, negative, or neutral**.  
2️⃣ **Language Detection** – Identifies the **language of the given text**.  
3️⃣ **Grammar & Spelling Correction** – Improves sentence structure and fixes grammatical errors.  

## 📌 Features  
✔️ **User Registration & Login** with secure authentication  
✔️ **Sentiment Analysis** using NLP Cloud API  
✔️ **Language Detection** from user input  
✔️ **Grammar & Spelling Correction** for better text formatting  
✔️ **API Integration** for real-time NLP processing
